# Hash
- Node : Key와 Value를 함께 저장. Key로 데이터를 저장하고 찾는다.
- Hashcode = Hash(Key)
- Index = Hashcode % N (N은 임의로 설정한다)

|Key       |Hashcode  |Index     |List              |
|----------|----------|----------|------------------|
|"Sung"    |445       |0         |jin -> hee -> min |
|"jin"     |321       |1         |sung              |
|"hee"     |306       |2         |                  |
|"min"     |324       |..        |                  |

- Hashcode를 거쳐 Index를 구하는데, Index를 잘 분산시키는 Hash가 좋은 Hash다.
